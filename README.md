# 📚 Daily Papers - CV / VLM / MedAI

每日精选计算机视觉、多模态大模型与医疗影像 AI 领域论文推荐。

🔗 **在线浏览**: https://jw-yuan.github.io/daily-papers-rec/

---

## 📂 内容分类

| 分类 | 说明 |
|------|------|
| 🏆 **顶会论文** | CVPR / ICCV / ECCV / ICML / ICLR / MICCAI / AAAI / NeurIPS 等 |
| 📘 **顶刊论文** | TPAMI / IJCV / TIP / TMI / MIA / JSHI 等 |
| 📄 **arXiv 最新** | cs.CV / cs.AI / eess.IV 每日更新 |

---

## 📝 笔记格式

每篇论文笔记包含以下字段：

- **核心贡献** - 论文的主要创新点
- **方法论** - 技术方法概述
- **实验结果** - 关键实验发现
- **局限性** - 论文自陈的不足
- **Abstract（原文）** - 英文原始摘要
- **摘要（中文翻译）** - 中文翻译

## 🔧 本地开发

```bash
# 克隆仓库
git clone https://github.com/JW-Yuan/daily-papers-rec.git
cd daily-paper-rec

# 启动本地服务器（任选其一）
python -m http.server 8000
# 或
npx serve .
```

然后访问 http://localhost:8000
