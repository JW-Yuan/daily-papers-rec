## 1. MambaBack: Bridging Local Features and Global Contexts in Whole Slide Image Analysis

| 项目 | 内容 |
|------|------|
| **作者** | Sicheng Chen, Chad Wong, Tianyi Zhang, Enhui Chai, Zeyu Liu, Fei Xia |
| **来源** | arXiv 2026 · cs.CV（2026-04-17） |
| **关键词** | Mamba混合架构, WSI分析, 局部特征与全局上下文桥接, Hilbert采样, 门控CNN, 多示例学习 |
| **原文** | https://arxiv.org/abs/2604.15729 |
| **架构图** | `papers/imgs/2026-04-23/MambaBack.png` |

### 核心贡献

MambaBack 提出融合 Mamba 与 MambaOut 优势的新型混合 MIL 骨干架构，系统解决现有 Mamba-MIL 方法的三大挑战：一维展平破坏二维空间局部性、对细粒度局部细胞结构建模不足、推理时内存峰值过高；通过 Hilbert 采样策略保留空间局部性、1D 门控 CNN 块捕获局部细胞特征、BiMamba2 块聚合全局上下文，在五个数据集上优于七种 SOTA 方法。

### 方法论

- **Hilbert 采样策略**：将 WSI 的二维 patch 布局按 Hilbert 曲线顺序展开为一维序列，在一维序列中最大程度保留二维空间局部性，使相邻 patch 在序列中依然保持空间邻近关系
- **1D 门控 CNN 模块（基于 MambaOut 思想）**：受 MambaOut 研究启发（SSM 在局部特征提取上是冗余的，门控 CNN 即可胜任），专用于提取细粒度局部细胞结构特征，计算效率高
- **BiMamba2 模块**：双向 Mamba2 模块负责全局上下文建模，利用 SSM 在 NLP 中验证的长程依赖捕获优势
- **非对称分块推理**：训练时并行处理，推理时采用分块-流式累积方式，显著降低边缘设备的峰值内存占用，提升部署可行性

### 实验结果

- 在 **5个数据集** 上优于 **7种 SOTA** 方法
- Hilbert 采样相比线性/光栅扫描策略保留更多空间局部信息，提升下游任务性能
- 非对称分块设计在不损失性能的前提下显著降低推理内存峰值
- 代码和数据集已公开，可复现性强

### 局限性

- Hilbert 曲线的最优分辨率需要针对不同放大倍率和 patch 大小调整
- 双模块混合架构（门控 CNN + BiMamba2）的深度和权重比例是额外超参数
- 当前主要验证分类任务，在检测和分割等密集预测任务中的性能需进一步验证

### Abstract

> Whole Slide Image (WSI) analysis is pivotal in computational pathology, enabling cancer diagnosis by integrating morphological and architectural cues across magnifications. Multiple Instance Learning (MIL) is the standard framework for WSI analysis. Recently, Mamba has emerged as a promising MIL backbone due to its efficiency and global context modeling inherited from NLP, outperforming Transformers. However, existing Mamba-based MIL methods face three key challenges: (1) destruction of 2D spatial locality during 1D sequence flattening, (2) suboptimal modeling of fine-grained local cellular structures, and (3) excessive memory peaks on resource-limited edge devices. Research shows (MambaOut) that Mamba's SSM component is redundant for extracting local features, where a Gated CNN suffices. Recognizing that WSI analysis requires both natural-image-like fine-grained local feature extraction and NLP-like global context modeling, we propose MambaBack, a novel hybrid architecture that combines the strengths of Mamba and MambaOut. MambaBack introduces a Hilbert sampling strategy to preserve 2D spatial locality, builds a hierarchical network with a 1D Gated CNN block to capture local cellular features and a BiMamba2 block to aggregate global context, and uses an asymmetric chunking design to allow parallel training while minimizing peak memory during inference. Experiments on five datasets demonstrate that MambaBack outperforms seven state-of-the-art methods. Source code and datasets are available.

### 摘要（中文）

WSI 分析是计算病理学的核心，Mamba 因其高效性和 NLP 传承的全局上下文建模能力成为 MIL 骨干的有力候选。然而，现有 Mamba-MIL 方法面临三大挑战：
1. 一维序列展平破坏二维空间局部性
2. 局部细胞结构建模效果欠佳
3. 边缘设备推理时内存峰值过高

研究表明（MambaOut），Mamba 的 SSM 组件在提取局部特征时是冗余的，使用门控 CNN 即可胜任。

提出 **MambaBack**，融合 Mamba 与 MambaOut 的优势：
- **Hilbert 采样**：在一维序列中保留切片的二维空间局部性
- **1D 门控 CNN 模块**：捕获局部细胞特征
- **BiMamba2 模块**：聚合全局上下文
- **非对称分块设计**：训练并行 + 推理分块流式，最小化峰值内存

在 **5个数据集** 上优于 **7种 SOTA** 方法。

## 2. ROAM: Region-Graph Optimal Transport Routing for Mixture-of-Experts Whole-Slide Image Classification

| 项目 | 内容 |
|------|------|
| **作者** | Xin Tian, Jiuliu Lu, Ephraim Tsalik, Bart Wanders, Colleen Knoth, Julian Knight |
| **来源** | arXiv 2026 · cs.CV / eess.IV（2026-04-08） |
| **关键词** | 混合专家MIL, 最优传输路由, 空间感知聚合, 容量约束, 图正则化, 全切片图像分类 |
| **原文** | https://arxiv.org/abs/2604.07298 |
| **架构图** | `papers/imgs/2026-04-23/ROAM.png` |

### 核心贡献

ROAM（Region-graph OptimAl-transport Mixture-of-experts）提出空间感知的 MoE-MIL 聚合器，通过容量约束的熵最优传输将区域 token 路由到专家池，从结构上强制均衡的专家利用率，解决无约束 softmax 路由导致专家退化为单路径的根本问题，同时通过图正则化 Sinkhorn 迭代引入空间连续性约束，在四个 WSI 基准及 NSCLC 跨域泛化任务（外部 AUC 0.845）上取得竞争性性能。

### 方法论

- **空间区域 Token 生成**：将密集 patch 包压缩为空间分箱单元，生成与局部组织邻域对齐的空间区域 token，捕获空间上下文信息
- **容量约束熵最优传输路由**：将区域到专家的分配建模为具有逐切片容量约束的熵最优传输（EOT）问题，通过 Sinkhorn 算法求解，无需辅助负载均衡损失即可强制均衡专家利用率
- **图正则化 Sinkhorn 迭代**：在空间区域图上扩散路由分配，鼓励相邻区域一致路由到相同专家，引入空间一致性先验
- **MoE-MIL 融合**：将混合专家（MoE）机制与 MIL 聚合器结合，使不同病理表型（如肿瘤、间质、坏死）由专门专家处理，提升跨异质切片的专业化能力

### 实验结果

- 在 **四个 WSI 基准数据集** 上与强 MIL 和 MoE 基线相比具有竞争力
- NSCLC 跨域泛化（TCGA→CPTAC）：外部 AUC **0.845 ± 0.019**，展现强泛化性
- 图正则化有效提升路由的空间一致性，减少相邻区域路由至不同专家的碎片化
- 无需辅助负载均衡损失即实现均衡专家利用，训练更稳定

### 局限性

- Sinkhorn 迭代在超大 WSI（patch 数量极多）时会增加计算开销
- 专家数量是额外超参数，影响模型容量和路由质量的权衡
- 当前验证主要在 NSCLC 场景，其他癌种（尤其异质性更高的癌种）的表现需进一步研究

### Abstract

> Multiple Instance Learning (MIL) is the dominant framework for computational pathology on gigapixel-scale whole-slide images (WSIs). However, current MIL aggregators route all instances through a shared pathway, limiting their capacity to specialize across the pathological heterogeneity within slides. Mixture-of-Experts (MoE) approaches offer a natural solution by partitioning instances to specialized expert sub-networks. Yet, unconstrained softmax routing can lead to highly imbalanced utilization — one or few experts absorbing the majority of routing mass, collapsing the mixture to a near-single-path solution. We introduce ROAM (Region-graph OptimAl-transport Mixture-of-experts), a spatially-aware MoE-MIL aggregator that routes region tokens to an expert pool via capacity-constrained entropic optimal transport, structurally promoting balanced expert utilization. We obtain spatial region tokens by compressing dense patch bags into spatially-binned units, aligning them with local tissue neighborhoods. Graph-regularized Sinkhorn iterations diffuse routing assignments over the spatial region graph, encouraging adjacent regions to route consistently to the same expert. Evaluated on four WSI benchmark datasets using frozen foundation model patch embeddings, ROAM achieves competitive performance against strong MIL and MoE baselines, with an external AUC of 0.845 ± 0.019 on NSCLC generalization (TCGA to CPTAC).

### 摘要（中文）

MIL 是计算病理学的主流框架，但当前 MIL 聚合器将所有实例通过共享路径路由，限制了跨切片内病理学异质性的专业化能力。混合专家（MoE）方法是自然解决方案，但无约束 softmax 路由会导致**专家利用率严重不均衡**，退化为近乎单路径的解决方案。

提出 **ROAM**（区域图最优传输混合专家）：
- **空间区域 Token**：将密集 patch 包压缩为空间分箱单元，与局部组织邻域对齐
- **容量约束熵最优传输路由**：将区域→专家分配建模为熵最优传输（EOT）问题，无需辅助损失强制均衡专家利用
- **图正则化 Sinkhorn 迭代**：在空间区域图上扩散路由分配，鼓励相邻区域一致路由

在 **四个 WSI 基准数据集** 上取得竞争性性能，NSCLC 跨域泛化（TCGA→CPTAC）外部 AUC **0.845 ± 0.019**。

## 3. Detecting Breast Carcinoma Metastasis on Whole-Slide Images by Partially Subsampled Multiple Instance Learning

| 项目 | 内容 |
|------|------|
| **作者** | Baichen Yu, Xuetong Li, Jing Zhou, Hansheng Wang |
| **来源** | arXiv 2026 · stat.ME / stat.AP（2026-04-19，Peking University） |
| **关键词** | 乳腺癌转移检测, 部分子采样MIL, 高斯混合模型, 最大似然估计, WSI, 鲁棒诊断 |
| **原文** | https://arxiv.org/abs/2604.17254 |

### 核心贡献

提出基于高斯混合的 MIL 框架，将部分子采样（Partially Subsampled MIL）的实例级预测建模为统计估计问题，通过基于袋的最大似然估计（BMLE）预测转移并引入子采样最大似然估计（SMLE）细化预测，在乳腺癌转移检测上超越 SOTA 且对模型误指定具有理论鲁棒性，为大规模 WSI 下 MIL 的统计基础提供了新视角。

### 方法论

- **高斯混合 MIL 建模**：将 WSI 中的 patch 特征建模为高斯混合分布，正例（转移区域）和负例（正常组织）各对应一个高斯分量，提供概率化的实例分类框架
- **基于袋的最大似然估计（BMLE）**：在 bag 级别而非实例级别进行最大似然估计，充分利用 bag 级标签信息，无需逐 patch 标注
- **子采样最大似然估计（SMLE）**：针对实际推理中只处理 WSI 子集 patch 的场景设计细化估计方法，利用子采样信息进一步提升预测精度
- **理论鲁棒性保证**：提供对模型误指定的鲁棒性分析，当实际分布偏离高斯混合假设时，方法仍保持合理的预测性能

### 实验结果

- 在乳腺癌转移检测任务上超越 SOTA 方法
- 对模型误指定具有鲁棒性，比纯深度学习方法具有更强的理论保证
- SMLE 相比 BMLE 进一步提升预测精度，利用子采样统计信息
- 方法具有可解释性，高斯混合参数提供疾病异质性的定量描述

### 局限性

- 高斯混合假设在形态学极度复杂的癌种上可能与实际分布存在较大偏差
- 最大似然估计在高维特征空间（如 ViT 提取的 768/1024 维特征）中的数值稳定性需关注
- 当前主要验证乳腺癌转移检测，向其他癌种和任务的泛化需进一步验证

### Abstract

> We propose a statistical framework for whole-slide image (WSI) classification based on partially subsampled multiple instance learning (MIL). Motivated by the challenge that WSIs often contain gigapixels requiring subsampling for practical computation, our approach models patch features as Gaussian mixture distributions with malignant and benign components. We introduce Bag-based Maximum Likelihood Estimation (BMLE) to predict metastasis using bag-level supervision, followed by Subsampled Maximum Likelihood Estimation (SMLE) to refine predictions by leveraging partially subsampled instances. Evaluated on breast carcinoma metastasis detection using whole-slide images, our method outperforms state-of-the-art approaches while maintaining robustness to model misspecification, providing a principled statistical foundation for large-scale WSI analysis under practical computational constraints.

### 摘要（中文）

提出基于部分子采样多示例学习（MIL）的 WSI 分类统计框架。由于 WSI 通常包含数十亿像素，实际计算中必须子采样，本方法将 patch 特征建模为高斯混合分布（恶性与良性分量分别对应一个高斯）。

- **BMLE（基于袋的最大似然估计）**：利用 bag 级监督预测转移
- **SMLE（子采样最大似然估计）**：通过利用部分子采样实例进一步细化预测

在乳腺癌转移检测的 WSI 任务上：
- 超越 SOTA 方法
- 对模型误指定具有**鲁棒性**
- 为实际计算约束下的大规模 WSI 分析提供了有原则的统计基础
