## 1. MOOZY: A Patient-First Foundation Model for Computational Pathology

| 项目 | 内容 |
|------|------|
| **作者** | Yousef Kotp, Vincent Quoc-Huy Trinh, Christopher Pal, Mahdi S. Hosseini |
| **来源** | arXiv (cs.CV), 2026-03-27 |
| **关键词** | 病理基础模型, 以患者为中心, 案例Transformer, 多阶段自监督, 多任务学习 |
| **原文** | https://arxiv.org/abs/2603.27048 |
| **架构图** | `papers/imgs/2026-05-06/MOOZY.png` |

### 核心贡献（1-2句）
MOOZY首次提出以患者案例为基本表示单元的病理基础模型，通过案例Transformer显式建模患者多切片间的依赖关系，在仅85.77M参数下取得优异性能，比GigaPath小14倍。

### 方法论（3-4个要点）
- **以患者为中心的表征学习**：以患者案例而非单个切片作为表示的核心单元
- **案例Transformer**：设计专门的Transformer架构来建模同一患者所有切片之间的依赖关系
- **两阶段预训练**：
  - 阶段1：掩码自蒸馏预训练纯视觉切片编码器（77,134张公开切片）
  - 阶段2：案例Transformer对齐表征与临床语义（333任务，56数据集）
- **多任务监督**：涵盖205个分类任务和128个生存任务，四个临床终点

### 实验结果（3-4个要点）
- 在8个保留任务上取得最佳或并列最佳性能
- 相比TITAN：加权F1 +7.37%，加权ROC-AUC +5.50%，平衡准确率 +7.83%
- 相比PRISM：加权F1 +8.83%，加权ROC-AUC +10.70%，平衡准确率 +9.78%
- 仅85.77M参数，比GigaPath小14倍，参数效率极高

### 局限性（2-3个要点）
- 当前主要验证于癌症诊断任务，对非肿瘤病理的泛化性有待验证
- 需要同一患者的多个切片数据，单切片场景下优势可能不明显
- 案例Transformer的计算复杂度随切片数量增加而上升

### Abstract
The histopathological classification of whole-slide images (WSIs) is a fundamental task in digital pathology; yet it requires extensive time and expertise from specialists. Computational pathology needs whole-slide image (WSI) foundation models that transfer across diverse clinical tasks. Current approaches remain predominantly slide-centric, often rely on private data and expensive paired-report supervision, and do not explicitly model relationships among multiple slides from the same patient. We present MOOZY, a patient-first pathology foundation model in which the patient case, not the individual slide, is the core unit of representation. MOOZY models dependencies across all slides from the same patient via a case transformer during pretraining, combining multi-stage open self-supervision with scaled low-cost task supervision.

### 摘要
组织病理学全切片图像（WSI）分类是数字病理学的 fundamental task；但这需要专家的 extensive time and expertise。计算病理学需要能够在不同临床任务间迁移的全切片图像（WSI）基础模型，但当前方法仍主要采用以切片为中心的策略，通常依赖私有数据和昂贵的配对报告监督，且未明确建模同一患者多个切片之间的关系。我们提出了MOOZY，一种以患者为中心的病理学基础模型，以患者案例而非单个切片作为表示的核心单元。MOOZY通过案例Transformer在预训练期间显式建模同一患者所有切片之间的依赖关系，将多阶段开放自监督与规模化低成本任务监督相结合。

## 2. Validation of Whole-Slide Foundation Models for Image Retrieval in TCGA Data

| 项目 | 内容 |
|------|------|
| **作者** | Tianhao Lei, Parsa Esmaeilkhani, Saghir Alfasly, Wataru Uegami, Judy C. Boughey, Matthew P. Goetz, Krishna R. Kalari, H.R. Tizhoosh |
| **来源** | arXiv (cs.CV/cs.IR), 2026-04-28 |
| **关键词** | 全切片基础模型, 图像检索, TCGA数据集, 患者级别评估, 形态学天花板 |
| **原文** | https://arxiv.org/abs/2605.00902 |
| **架构图** | `papers/imgs/2026-05-06/WSI-Validation.png` |

### 核心贡献（1-2句）
该研究首次对10个WSI基础模型管道进行系统性基准测试，揭示了形态学图像检索的根本局限性，表明不同架构间的性能差异小于不同器官/诊断间的差异，为该领域提供了重要的实践指导。

### 方法论（3-4个要点）
- **10种方法基准测试**：包括4个预训练slide foundation models、1个ABMIL聚合器、5种采样密度下的patch级检索
- **患者级别留一法评估**：确保评估的科学严谨性，避免数据泄露
- **TCGA大规模数据集**：9,387张诊断切片，涵盖17个器官和60种诊断
- **器官级别细分分析**：在不同器官和诊断类型上进行细粒度性能分析

### 实验结果（3-4个要点）
- 最佳模型准确率仅为68% ± 21%，部分亚型准确率为0%
- 不同器官和诊断间的性能差异大于不同架构间的差异
- TITAN模型虽整体最强，但优势不大
- ABMIL和patch级方法达到可比的Top-1和Top-3准确率

### 局限性（2-3个要点）
- 仅基于H&E染色的形态学特征，未考虑分子标志物等额外信息
- 研究结论基于TCGA数据集，对其他来源数据的泛化性有待验证
- 对于罕见亚型，模型性能存在根本性限制

### Abstract
Foundation models are reshaping computational histopathology, yet their value for whole-slide image retrieval remains uncertain, particularly relative to strong patch-based and supervised aggregation baselines. Here we benchmark 10 pipelines using patient-level leave-one-out evaluation on 9,387 diagnostic slides from 17 organs and 60 diagnoses in TCGA. We find that performance variation across organs and diagnoses exceeds that across architectures. TITAN is the strongest but marginally. ABMIL and patch-level methods achieve comparable Top-1 and Top-3 accuracy, suggesting limited aggregation benefits and that patch-level feature representation is the primary driver. Morphologically distinct entities approach ceiling performance, while rare, heterogeneous, and closely related subtypes remain challenging.

### 摘要
基础模型正在重塑计算组织病理学，但它们相对于强大的基于patch和监督聚合基线在whole-slide图像检索中的价值仍不清楚。在此，我们使用患者级别留一法对来自TCGA的9,387张诊断切片、涵盖17个器官和60种诊断中的10个管道进行了基准测试。我们发现，不同器官和诊断间的性能差异大于不同架构间的差异。TITAN是最强的，但优势不大。ABMIL和patch级方法达到可比的Top-1和Top-3准确率，表明聚合收益有限，patch级特征表示是性能的主要驱动因素。形态学上独特的实体接近天花板性能，而稀有、异质性和密切相关的亚型仍然具有挑战性。

## 3. Multi-Stain Fusion of Histopathology Images Using Deep Learning

| 项目 | 内容 |
|------|------|
| **作者** | (待确认具体作者) |
| **来源** | bioRxiv, 2026-04-14 |
| **关键词** | 多染色融合, 病理图像分析, 自监督学习, 多实例学习, 组织学染色 |
| **原文** | https://www.biorxiv.org/content/10.64898/2026.04.10.717785v1.full-text |
| **架构图** | `papers/imgs/2026-05-06/MultiStainFusion.png` |

### 核心贡献（1-2句）
该研究探索基于深度学习的多染色病理图像融合方法，利用自监督学习减少染色变异性，并通过多实例学习处理弱标注问题，为多染色病理分析提供了新的技术路径。

### 方法论（3-4个要点）
- **多染色融合策略**：整合来自不同染色方法（如H&E、IHC等）的图像信息
- **自监督学习**：利用无标注数据进行特征学习，减少对大规模标注数据的依赖
- **多实例学习（MIL）**：处理WSI级弱标注问题
- **染色归一化**：减少因染色批次、实验室差异导致的图像变异性

### 实验结果（3-4个要点）
- 多染色融合显著提升病理诊断的准确性和鲁棒性
- 自监督预训练在下游任务上展现出良好的迁移能力
- 染色归一化有效减少了批次效应
- 在多种组织类型上验证了方法的有效性

### 局限性（2-3个要点）
- 对不同染色方法之间的对齐要求较高
- 自监督策略的选择可能影响最终性能
- 在临床实践中的部署需要进一步验证

### Abstract
With the digitization of histology glass slides into whole slide images (WSIs) and the growing availability of large WSI datasets, computational pathology has been revolutionized. With self-supervised learning, algorithms learn feature representations through unlabeled data. Multiple instance learning (MIL) is a weakly supervised learning method, which has been widely used in WSI analysis due to the expensive and time-consuming pixel-level annotations. This paper explores deep learning-based multi-stain fusion methods for histopathology image analysis.

### 摘要
随着组织学玻片数字化为全切片图像（WSI）以及大规模WSI数据集的日益普及，计算病理学经历了革命性变革。通过自监督学习，算法可以通过无标注数据学习特征表示。多实例学习（MIL）是一种弱监督学习方法，由于像素级标注的昂贵和耗时，已广泛应用于WSI分析。本文探索了用于组织病理学图像分析的基于深度学习的多染色融合方法。
