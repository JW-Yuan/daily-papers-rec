### 1. CoME-VL: Scaling Complementary Multi-Encoder Vision-Language Learning

| 项目 | 内容 |
|------|------|
| **作者** | Ankan Deria, Komal Kumar, Xilin He, Imran Razzak, Hisham Cholakkal, Fahad Shahbaz Khan, Salman Khan |
| **来源** | arXiv:2604.03231 (cs.CV) · 2026-04-03 · MBZUAI |
| **原文** | https://arxiv.org/abs/2604.03231 |
| **代码** | https://github.com/mbzuai-oryx/CoME-VL |
| **架构图** | `papers/imgs/2026-04-07/CoME-VL.png` |

#### 🔑 核心贡献
提出了 CoME-VL，一种模块化融合框架，将对比训练视觉编码器（SigLIP）与自监督 DINO 编码器集成到视觉语言模型中。通过基于熵的多层特征聚合（结合正交约束投影）减少冗余，并利用 RoPE 增强的交叉注意力对齐异构 token 网格，生成紧凑的融合视觉 token，在多项 VLM 基准上显著优于单编码器基线。

#### 🔬 方法论
- **双编码器融合**：SigLIP（对比学习）提供语义对齐能力，DINO（自监督）提供丰富密集语义
- **熵引导多层聚合**：基于熵选择最信息量的特征层，结合正交正则化约束减少特征冗余
- **RoPE 增强交叉注意力**：在空间上对齐两个编码器输出的异构 token 网格，生成紧凑融合 token
- **最小侵入性设计**：融合后的 token 可无缝注入标准仅解码器 LLM，无需修改语言模型结构

#### 📊 实验结果
- 视觉理解任务平均提升 **4.9%**
- 定位任务（Grounding）平均提升 **5.4%**
- 在 RefCOCO 检测任务上达到 SOTA 性能
- 多个 VLM 基准上一致优于单编码器基线方法

#### ⚠️ 局限性
- 双编码器推理的计算成本高于单编码器架构
- 融合 token 压缩过程可能损失部分精细空间信息
- 在视频理解等时序任务上的效果有待验证

#### 📝 Abstract（原文）
> Recent vision-language models (VLMs) typically rely on a single visual encoder trained with contrastive image-text objectives, such as CLIP-style pretraining. While contrastive encoders excel at cross-modal alignment and retrieval, self-supervised visual encoders tend to capture richer dense semantics and exhibit stronger robustness in recognition and understanding tasks. In this paper, we investigate how to fuse and scale these complementary visual representations for vision-language modeling. We propose CoME-VL: a modular fusion framework that integrates a contrastively trained visual encoder with a self-supervised DINO encoder. Our approach achieves representation-level fusion via (i) entropy-based multi-layer aggregation with orthogonal constraint projection to reduce redundancy, and (ii) RoPE-enhanced cross-attention to align heterogeneous token grids and generate compact fused visual tokens. The fused tokens can be injected into decoder-only LLMs with minimal modifications, staying compatible with standard VLM pipelines. Extensive experiments on multiple vision-language benchmarks demonstrate that CoME-VL consistently outperforms single-encoder baselines, specifically achieving an average improvement of 4.9% on visual understanding tasks and 5.4% on grounding tasks. Our approach achieves state-of-the-art performance on RefCOCO detection tasks, with significant improvements relative to the baseline.

#### 📝 摘要（中文翻译）
近期视觉语言模型（VLM）通常依赖单一视觉编码器，并通过对比式图像-文本目标（如 CLIP 风格预训练）进行训练。对比式编码器在跨模态对齐和检索方面表现优异，而自监督视觉编码器往往能捕捉更丰富的密集语义，并在识别与理解任务上展现出更强的鲁棒性。本文研究如何将这两种互补的视觉表示进行融合并扩展用于视觉语言建模。我们提出了 CoME-VL：一种模块化融合框架，它将对比训练视觉编码器与自监督 DINO 编码器集成在一起。我们的方法通过（i）基于熵的多层聚合与正交约束投影来减少冗余，以及（ii）RoPE 增强的交叉注意力来对齐异构 token 网格并生成紧凑的融合视觉 token，实现表示级融合。融合后的 token 可以以最小的改动注入到仅解码器的 LLM 中，与标准 VLM 流程兼容。在多个视觉语言基准测试上的广泛实验表明，CoME-VL 一致优于单编码器基线，具体而言，在视觉理解任务上平均提升 4.9%，在定位任务上平均提升 5.4%。我们的方法在 RefCOCO 检测任务上达到了最先进的性能，同时相对基线有大幅提升。

### 2. MI-Pruner: Crossmodal Mutual Information-guided Token Pruner for Efficient MLLMs

| 项目 | 内容 |
|------|------|
| **作者** | Jiameng Li, Aleksei Tiulpin, Matthew B. Blaschko |
| **来源** | arXiv:2604.03072 (cs.CV) · 2026-04-03 |
| **关键词** | Token剪枝, 互信息, 多模态大模型, 跨模态依赖, 推理加速 |
| **原文** | https://arxiv.org/abs/2604.03072 |
| **架构图** | `papers/imgs/2026-04-07/MI-Pruner.png` |

#### 🔑 核心贡献
提出了 MI-Pruner，一种基于跨模态互信息（Mutual Information）的视觉 token 剪枝方法，用于提升多模态大语言模型（MLLM）的推理效率。不同于依赖注意力分数的传统方法，MI-Pruner 直接在视觉特征与文本特征交互之前计算两者的互信息，实现更精确的跨模态依赖度量，无需访问内部注意力图或修改模型架构。

#### 🔬 方法论
- 在视觉-文本特征交互发生之前，直接计算两者间的互信息（MI）
- 基于互信息显式度量视觉 token 对当前文本查询的贡献度
- 方法非侵入性：无需访问注意力图或修改模型架构
- 适用于任意多模态大语言模型，具有即插即用特性

#### 📊 实验结果
- 在多项 MLLM 基准测试上优于基于注意力分数的 token 剪枝方法
- 引入延迟极小，在效率与性能间取得更优平衡
- 无需重训练即可应用于现有 MLLM

#### ⚠️ 局限性
- 互信息估计的计算方式对特征分布的假设有一定敏感性
- 在高分辨率输入（如医学图像）场景下的剪枝效果需进一步验证
- 当前评估主要集中在通用 VQA 任务，医学等专业领域有待探索

#### 📝 Abstract（原文）
> For multimodal large language models (MLLMs), visual information is relatively sparse compared with text. As a result, research on visual pruning emerges for efficient inference. Current approaches typically measure token importance based on the attention scores in the visual encoder or in the LLM decoder, then select visual tokens with high attention scores while pruning others. In this paper, we pursue a different and more surgical approach. Instead of relying on mechanism-specific signals, we directly compute Mutual Information (MI) between visual and textual features themselves, prior to their interaction. This allows us to explicitly measure crossmodal dependency at the feature levels. Our MI-Pruner is simple, efficient and non-intrusive, requiring no access to internal attention maps or architectural modifications. Experimental results demonstrate that our approach outperforms previous attention-based pruning methods with minimal latency.

#### 📝 摘要（中文翻译）
对于多模态大语言模型（MLLM），与文本相比，视觉信息相对稀疏。因此，针对高效推理的视觉剪枝研究应运而生。当前方法通常基于视觉编码器或 LLM 解码器中的注意力分数来衡量 token 的重要性，然后选择高注意力分数的视觉 token 并剪枝其余 token。本文采用一种不同且更精确的方法。与依赖特定机制信号不同，我们在视觉特征和文本特征交互之前，直接计算两者之间的互信息（MI）。这使我们能够在特征层面显式度量跨模态依赖性。MI-Pruner 简单、高效且非侵入式，无需访问内部注意力图或修改架构。实验结果表明，我们的方法以极小的延迟代价优于之前基于注意力的剪枝方法。

### 3. PR3DICTR: A Modular AI Framework for Medical 3D Image-based Detection and Outcome Prediction

| 项目 | 内容 |
|------|------|
| **作者** | Daniel C. MacRae, Luuk van der Hoek, Robert van der Wal, Suzanne P. M. de Vette, Hendrike Neh, Baoqiang Ma, Peter M. A. van Ooijen, Lisanne V. van Dijk |
| **来源** | arXiv:2604.03203 (cs.CV, eess.IV) · 2026-04-03 |
| **关键词** | 3D医学图像, 分类框架, PyTorch, MONAI, 模块化设计, 预测模型 |
| **原文** | https://arxiv.org/abs/2604.03203 |

#### 🔑 核心贡献
提出了 PR3DICTR（3D Image Classification and Standardized Training Research Platform），一个基于 PyTorch 和 MONAI 社区标准的开放访问、灵活便捷的医学 3D 图像预测模型开发框架。通过模块化设计原则和标准化，减轻开发负担的同时保留充分的可调整性，仅需最少两行代码即可运行，支持任何基于二元或事件的 3D 分类任务。

#### 🔬 方法论
- 基于 PyTorch 和 MONAI 构建，符合医学影像 AI 社区标准
- 模块化架构设计：支持自定义插入用户自己的模型、数据处理和训练模块
- 提供丰富预置功能：模型架构选项、超参数配置和训练方法
- 极简使用体验：最少两行代码即可完成 3D 医学图像分类任务配置与运行

#### 📊 实验结果
- 框架已在多种 3D 医学图像分类任务上验证可用性
- 支持二元分类和事件预测等多种任务形式
- 与现有 MONAI 生态系统无缝集成，可直接利用丰富的医学预训练模型
- 开放访问，可显著降低医学影像 AI 研究的技术门槛

#### ⚠️ 局限性
- 当前主要面向分类任务，分割和检测等任务的原生支持有限
- 框架的灵活性与易用性之间仍存在一定权衡
- 在超大规模数据集和分布式训练场景下的性能优化有待验证

#### 📝 Abstract（原文）
> 3D medical imaging data and computer-aided decision making, particularly deep learning, are becoming increasingly important in medicine. To support these developments, we introduce PR3DICTR: 3D Image Classification and Standardized Training Research Platform. Built on community standard releases (PyTorch and MONAI), it provides an open-access, flexible and convenient framework for the development of predictive models, with particular focus on classification using 3D medical imaging data. By combining modular design principles with standardization, it aims to reduce the development burden while retaining adjustability. It provides users with a wide range of out-of-the-box features such as model architecture design options, hyperparameter solutions and training methods, while still providing the opportunity and freedom for users to "plug in" their own solutions or modules. PR3DICTR can be applied to any binary or event-based 3D classification task, and requires a minimum of two lines of code to run.

#### 📝 摘要（中文翻译）
三维医学影像数据和计算机辅助决策（尤其是深度学习）在医学领域变得越来越重要。为了支持这些发展，我们介绍了 PR3DICTR：三维图像分类与标准化训练研究平台。该平台基于社区标准发行版（PyTorch 和 MONAI）构建，提供了一个开放访问、灵活且便捷的预测模型开发框架，特别侧重于使用三维医学影像数据进行分类。通过结合模块化设计原则和标准化，它旨在减轻开发负担，同时保留可调整性。它为用户提供了大量预置功能，例如模型架构设计选项、超参数解决方案和训练方法，但仍为用户提供了"插入"自己解决方案或模块的机会和自由。PR3DICTR 可应用于任何基于二元或事件的三维分类任务，并且最少仅需两行代码即可运行。
