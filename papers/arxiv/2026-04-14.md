## 1. ROAM: Region-Graph Optimal Transport Routing for Mixture-of-Experts Whole-Slide Image Classification

| 项目 | 内容 |
|------|------|
| **作者** | Xin Tian, Jiuliu Lu, Ephraim Tsalik, Bart Wanders, Colleen Knoth, Julian Knight |
| **来源** | arXiv: 2604.07298 (2026-04-08, cs.CV / cs.AI / eess.IV) |
| **关键词** | multiple instance learning, mixture of experts, optimal transport, whole slide image, spatial aggregation, Sinkhorn |
| **原文** | https://arxiv.org/abs/2604.07298 |
| **架构图** | `papers/imgs/2026-04-14/ROAM.png` |

#### 🔑 核心贡献

ROAM 提出将**容量约束熵最优传输（entropic optimal transport）**引入 MoE-MIL 聚合器，通过 Sinkhorn 算法实现空间感知的区域-专家路由，无需辅助负载均衡损失即可保证专家利用率均衡，有效处理 WSI 中固有的病理异质性。

#### 🔬 方法论

- **空间区域化**：将密集 patch 袋压缩为空间分箱的区域 token，每个区域 token 代表局部组织邻域的聚合表示，使路由决策与病理结构空间分布对齐
- **熵最优传输路由**：使用 Sinkhorn 算法对区域 token 到专家（Pooler）进行容量约束分配，容量边际设计保证每个专家处理相近数量的区域，避免专家塌陷（expert collapse）问题
- **图正则化 Sinkhorn 迭代**：在空间区域邻接图上扩散路由分配矩阵，鼓励相邻区域路由至相同专家，使专家学习空间上连贯的组织表型（如肿瘤床、浸润区等）
- **冻结基础模型兼容**：ROAM 聚合器直接作用于预提取的冻结 patch 嵌入，可无缝集成至 UNI、CONCH 等任何现有病理基础模型的 MIL 流水线

#### 📊 实验结果

- 在 NSCLC 类型分类的 TCGA→CPTAC 外部泛化评估中，AUC 达 **0.845 ± 0.019**，展现优异跨数据集泛化能力
- 在 4 个 WSI 分类基准数据集上，使用冻结基础模型 patch 嵌入，性能与强 MIL/MoE 基线具有竞争力
- 图正则化 Sinkhorn 相比无正则版本显著提升路由的空间一致性，消融实验验证各组件贡献
- 专家利用率分析表明，ROAM 实现真正的平衡路由，各专家激活率差异极小

#### ⚠️ 局限性

- 区域分箱大小作为超参数需调优，粒度设置影响路由质量和计算开销
- Sinkhorn 迭代引入额外计算成本，在超大 WSI（patch 数>50K）场景下效率有待优化
- 专家数量的选择对最终性能影响较大，最优配置因任务和数据集而异

#### 📝 Abstract（原文）

Multiple Instance Learning (MIL) is the dominant framework for gigapixel whole-slide image (WSI) classification in computational pathology. However, current MIL aggregators route all instances through a shared pathway, limiting specialization for the inherent pathological heterogeneity in each slide. Mixture-of-Experts (MoE) methods naturally address this by assigning instances to specialized expert sub-networks, but unconstrained softmax routing can lead to highly imbalanced utilization — one or few experts absorb most routing weight, collapsing the mixture toward a near single-path solution. We propose ROAM (Region-graph OptimAl-transport Mixture-of-experts), a spatially aware MoE-MIL aggregator that routes region tokens to expert poolers via capacity-constrained entropic optimal transport, enforcing balanced expert utilization by design without auxiliary load-balancing losses. Region tokens are obtained by compressing dense patch bags into spatially-binned units, aligning routing with local tissue neighborhoods. A graph-regularized Sinkhorn iteration then diffuses routing assignments over a spatial region graph, encouraging adjacent regions to route consistently to the same expert. We evaluate ROAM on four WSI benchmarks using frozen foundation model patch embeddings, showing competitive performance against strong MIL and MoE baselines, and achieving 0.845 ± 0.019 external AUC on TCGA-CPTAC NSCLC generalization.

#### 📝 摘要（中文翻译）

多示例学习（MIL）是计算病理学中全切片图像（WSI）分类的主流框架。然而，当前 MIL 聚合器将所有实例通过共享路径路由，限制了对每张切片固有病理异质性的专业化处理。混合专家（MoE）方法通过将实例分配给专业化专家子网络来自然解决这一问题，但无约束的 softmax 路由可能导致高度不平衡的利用率——一个或少数专家吸收大部分路由权重，使混合模型退化为接近单路径的解。我们提出 ROAM（区域图最优传输混合专家），一种空间感知的 MoE-MIL 聚合器，通过容量约束的熵最优传输将区域 token 路由到专家池化器，从设计上保证专家利用率均衡，无需辅助负载均衡损失。区域 token 通过将密集 patch 袋压缩为空间分箱单元获得，使路由与局部组织邻域对齐。图正则化 Sinkhorn 迭代在空间区域图上扩散路由分配，鼓励相邻区域一致路由到相同专家。在四个 WSI 基准数据集上，使用冻结基础模型 patch 嵌入评估，ROAM 与强 MIL/MoE 基线相比具有竞争力，在 TCGA-CPTAC NSCLC 泛化评估中外部 AUC 达 0.845 ± 0.019。

## 2. MorphDistill: Distilling Unified Morphological Knowledge from Pathology Foundation Models for Colorectal Cancer Survival Prediction

| 项目 | 内容 |
|------|------|
| **作者** | Hikmat Khan, Usama Sajjad, Metin N. Gurcan, Anil Parwani, Wendy L. Frankel, Wei Chen, Muhammad Khalid Khan Niazi |
| **来源** | arXiv: 2604.06390 (2026-04-07, cs.CV / cs.AI) |
| **关键词** | knowledge distillation, pathology foundation model, colorectal cancer, survival prediction, MIL, multi-teacher distillation |
| **原文** | https://arxiv.org/abs/2604.06390 |

#### 🔑 核心贡献

MorphDistill 提出**多教师关系蒸馏策略**，从 10 个病理基础模型中并行蒸馏样本间关系知识，训练出针对结直肠癌（CRC）的特定形态学编码器，在标准 TCGA 和外部 Alliance/CALGB 验证队列上显著提升生存预测准确性，无需显式特征空间对齐。

#### 🔬 方法论

- **维度无关多教师蒸馏（Stage I）**：从 10 个病理基础模型（具有不同特征维度）中同时蒸馏，采用维度无关的关系知识蒸馏，保留样本间相似性结构而非显式特征向量对齐，配合监督对比正则化训练 CRC 专属学生编码器
- **注意力 MIL 聚合（Stage II）**：从全切片图像提取 patch 级特征后，通过注意力 MIL 模型聚合切片级表示，最终预测五年生存率
- **大规模 CRC 预训练数据**：在大规模结直肠数据集（无需分割标注）上执行第一阶段蒸馏，学习器官特异性形态学先验
- **无基因组数据依赖**：纯视觉路径，无需配对的基因组/转录组数据，临床部署门槛低

#### 📊 实验结果

- Alliance/CALGB 89803 队列（III 期 CRC, n=424）：AUC **0.68**，C-index **0.661**，风险比 **2.52**（95% CI: 1.73–3.65），相比最强基线（AUC 0.63）相对提升约 8%
- TCGA 外部验证（n=562）：C-index **0.628**，展示强泛化能力
- 多教师蒸馏显著优于单一基础模型特征提取，证明集成多样化形态学知识的优势
- 在极小患者队列上仍取得稳定性能，适合罕见亚型的生存建模

#### ⚠️ 局限性

- C-index 和 AUC 绝对数值仍有提升空间，尤其在外部验证队列上
- 蒸馏效果受限于教师模型的多样性，若多个教师编码相似特征则收益递减
- 仅针对结直肠癌设计，向其他癌种迁移需重新执行第一阶段蒸馏

#### 📝 Abstract（原文）

Background: Colorectal cancer (CRC) remains a leading cause of cancer-related mortality worldwide. Accurate survival prediction is essential for treatment stratification, yet existing pathology foundation models often overlook organ-specific features critical for prognosis. Methods: We propose MorphDistill, a two-stage framework. In Stage I, we train a student encoder using dimension-agnostic multi-teacher relational distillation from 10 foundation models combined with supervised contrastive regularization on a large-scale colorectal dataset, preserving inter-sample relationships without explicit feature alignment. In Stage II, we extract patch-level features from WSIs and aggregate them via attention-based MIL to predict five-year survival. Results: On the Alliance/CALGB 89803 cohort (n=424, Stage III CRC), MorphDistill achieves AUC 0.68 (SD 0.08), C-index 0.661, and hazard ratio 2.52 (95% CI: 1.73–3.65), approximately 8% relative improvement over the strongest baseline. External validation on TCGA (n=562) yields C-index 0.628, demonstrating strong generalizability. Conclusion: MorphDistill offers an efficient strategy for task-specific representation learning in computational pathology prognosis.

#### 📝 摘要（中文翻译）

背景：结直肠癌（CRC）是全球癌症相关死亡的主要原因之一。准确的生存预测对治疗分层至关重要，然而现有病理基础模型往往忽视了对预后至关重要的器官特异性特征。方法：我们提出 MorphDistill，一种两阶段框架。第一阶段，在大规模结直肠数据集上，通过来自 10 个基础模型的维度无关多教师关系蒸馏结合监督对比正则化训练学生编码器，在无需显式特征对齐的情况下保留样本间关系。第二阶段，从全切片图像提取 patch 级特征，通过注意力 MIL 聚合后预测五年生存率。结果：在 Alliance/CALGB 89803 队列（III 期 CRC，n=424）上，MorphDistill 实现 AUC 0.68，C-index 0.661，风险比 2.52（95% CI: 1.73–3.65），相比最强基线相对提升约 8%。在 TCGA 外部验证（n=562）上 C-index 达 0.628，展示强泛化能力。结论：MorphDistill 为计算病理学预后建模中的任务特定表示学习提供了一种高效策略。

## 3. Cluster-First Labelling: An Automated Pipeline for Segmentation and Morphological Clustering in Histology Whole Slide Images

| 项目 | 内容 |
|------|------|
| **作者** | Muhammad Haseeb Ahmad, Sharmila Rajendran, Damion Young, Jon Mason |
| **来源** | arXiv: 2604.09370 (2026-04-10, q-bio.QM / cs.CV) · University of Oxford |
| **关键词** | histopathology annotation, morphological clustering, Cellpose-SAM, DBSCAN, UMAP, active learning, WSI |
| **原文** | https://arxiv.org/abs/2604.09370 |
| **代码** | https://github.com/OxfordCompetencyCenters/edu06_histology_labelling |

#### 🔑 核心贡献

提出**聚类优先（Cluster-First）**自动化标注范式，通过 Cellpose-SAM 全自动分割、ResNet-50 特征嵌入、UMAP 降维和 DBSCAN 聚类将形态相似结构分组，人工仅需标注代表性聚类而非单个对象，将病理 WSI 标注工作量降低数量级，在 3696 个组织成分上实现 96.8% 的聚类-标签对齐准确率。

#### 🔬 方法论

- **全自动分割（Cellpose-SAM）**：集成 Cellpose 与 Segment Anything Model，对 WSI tile 中的细胞/细胞核等组织成分执行精细实例分割，无需手动标注训练数据
- **神经嵌入提取**：使用预训练 ResNet-50 从每个分割出的组织成分提取高维特征嵌入，捕获形态学外观的深层语义
- **UMAP + DBSCAN 无监督聚类**：通过 UMAP 将高维嵌入映射至低维流形，DBSCAN 在低维空间聚类，自动发现形态相似的组织成分组，无需预定义类别数
- **云原生标注应用**：提供配套的 Web 标注应用，展示聚类分组后供病理学家对代表性聚类（而非单个对象）进行批量标注，大幅加速标注流程

#### 📊 实验结果

- 在 3696 个组织成分（来自人类、大鼠、兔子 3 物种，13 种组织类型）上测试，加权聚类-标签对齐准确率达 **96.8%**
- 13 种组织类型中有 7 种实现完美匹配（Hungarian 算法对齐），展示方法的泛物种、跨组织类型适用性
- 标注工作量相比逐个对象标注降低数量级，在 10000+ 组织成分的 WSI 上尤为显著
- 代码、标注应用、评估代码全部开源，支持可重复研究与工业级部署

#### ⚠️ 局限性

- Cellpose-SAM 分割质量影响后续聚类的可靠性，复杂病理样本（如低分化肿瘤）分割误差可能较大
- DBSCAN 对密度参数敏感，形态高度多样的样本集群划分可能不稳定
- 当前仅在常规组织学染色（H&E）上验证，IHC、特殊染色等多模态场景适用性待评估

#### 📝 Abstract（原文）

Pathology whole slide images (WSIs) can contain tens of thousands of structures — cells, nuclei, and other morphologically diverse objects — each of which must be individually annotated and classified by expert pathologists. This labour-intensive process has become a bottleneck in computational pathology. We propose an alternative paradigm: rather than labelling cells one at a time, our pipeline first segments all cells automatically using Cellpose-SAM, then extracts neural embeddings via a pretrained ResNet-50, reduces dimensionality using UMAP, and clusters morphologically similar objects using DBSCAN. Human annotators then only need to label representative clusters rather than individual objects, reducing labelling effort by orders of magnitude. We evaluate our approach on 3,696 tissue components across 13 tissue types from 3 species (human, rat, rabbit), achieving a weighted cluster-to-label alignment accuracy of 96.8%, with 7 of 13 tissue types achieving perfect matches using the Hungarian algorithm. All pipeline code, annotation web application, and evaluation code are released as open-source resources.

#### 📝 摘要（中文翻译）

病理全切片图像（WSI）可能包含数万个结构——细胞、细胞核及其他形态各异的对象——每个都需要专业病理学家逐一标注和分类。这一劳动密集型过程已成为计算病理学的瓶颈。我们提出一种替代范式：与其逐一标注细胞，不如先用 Cellpose-SAM 自动分割所有细胞，再通过预训练 ResNet-50 提取神经嵌入，用 UMAP 降维后以 DBSCAN 聚类形态相似对象。人工标注者只需标注代表性聚类而非单个对象，将标注工作量降低数量级。我们在来自 3 物种（人类、大鼠、兔子）的 13 种组织类型共 3696 个组织成分上评估，实现加权聚类-标签对齐准确率 96.8%，13 种组织类型中 7 种通过匈牙利算法实现完美匹配。所有流水线代码、标注 Web 应用和评估代码均作为开源资源发布。
