### 1. AEM: Attention Entropy Maximization for Multiple Instance Learning based Whole Slide Image Classification

| 项目 | 内容 |
|------|------|
| **作者** | Yunlong Zhang, Honglin Li, Yunxuan Sun, Zhongyi Shui, Jingxiong Li, Chenglu Zhu, Lin Yang |
| **来源** | MICCAI 2025 · arXiv:2406.15303 |
| **关键词** | 多示例学习, 全切片图像分类, 注意力熵最大化, MIL正则化, 过拟合缓解 |
| **原文** | https://arxiv.org/abs/2406.15303 |
| **代码** | https://github.com/dazhangyu123/AEM |
| **架构图** | `papers/imgs/2026-04-13/AEM.png` |

#### 🔑 核心贡献
提出 **AEM（Attention Entropy Maximization，注意力熵最大化）**，一种简单而有效的正则化技术，用于缓解多示例学习（MIL）在全切片图像（WSI）分类中因注意力过度集中而导致的过拟合问题。AEM 发现注意力熵与模型性能之间存在正相关关系，通过惩罚过度的注意力集中来改善 MIL 性能，并引入余弦权重退火解决超参数敏感性问题。在多种特征提取器、MIL 框架和注意力机制下验证了 AEM 的优越性能。

#### 🔬 方法论
- **注意力熵最大化正则化**：通过最大化注意力分布的熵来惩罚过度的注意力集中，强制模型关注更多样化的实例
- **余弦权重退火（Cosine Weight Annealing）**：动态调整 AEM 正则化权重，在训练初期给予模型更大灵活性，逐步收紧约束
- **通用兼容性**：无需修改 MIL 框架或注意力机制，可直接集成到现有 MIL 聚合器中
- **无需额外数据或监督**：纯正则化方法，不依赖辅助标注信息

#### 📊 实验结果
- 在多种 WSI 分类基准数据集上均取得领先性能，验证了方法的有效性和通用性
- 注意力熵与分类性能的正相关关系在多个数据集上得到验证
- 与复杂架构修改相比，AEM 以更简单的实现取得相当或更优的性能
- 余弦退火策略有效缓解了 AEM 权重选择的敏感性

#### ⚠️ 局限性
- 熵最大化对极端不平衡数据集（正例极少）的效果可能受限
- 当前验证主要集中在分类任务，对分割等密集预测任务的适用性有待评估
- 超参数（如初始权重、衰减速率）对不同数据集的最优值可能不同

#### 📝 Abstract（原文）
> Multiple Instance Learning (MIL) effectively analyzes whole slide images but faces overfitting due to attention over-concentration. While existing solutions rely on complex architectural modifications or additional processing steps, we propose Attention Entropy Maximization (AEM), a lightweight yet powerful approach for mitigating attention concentration and improving MIL performance. AEM directly maximizes the entropy of the attention distribution to penalize excessive concentration. Furthermore, we observe a positive correlation between attention entropy and model performance and introduce cosine weight annealing to address the sensitivity of AEM's weight parameter. Comprehensive experiments on multiple feature extractors, MIL frameworks, and attention mechanisms validate the effectiveness of AEM.

#### 📝 摘要（中文翻译）
多示例学习（MIL）能有效分析全切片图像，但面临因注意力过度集中而导致的过拟合问题。现有解决方案依赖复杂的架构修改或额外处理步骤，本文提出注意力熵最大化（AEM），一种轻量级但有效的方法，用于缓解注意力集中并提升 MIL 性能。AEM 直接最大化注意力分布的熵以惩罚过度集中。此外，我们发现注意力熵与模型性能之间存在正相关关系，并引入余弦权重退火来解决 AEM 权重参数的敏感性问题。在多种特征提取器、MIL 框架和注意力机制上的综合实验验证了 AEM 的有效性。

### 2. PathVQ: Reforming Computational Pathology Foundation Model for Whole Slide Image Analysis via Vector Quantization

| 项目 | 内容 |
|------|------|
| **作者** | Honglin Li, Zhongyi Shui, Yunlong Zhang, Chenglu Zhu, Lin Yang |
| **来源** | ICCV 2025 · arXiv:2503.06482 |
| **关键词** | 向量量化, 全切片图像分析, 病理基础模型, Token压缩, 自监督学习, 多尺度量化 |
| **原文** | https://arxiv.org/abs/2503.06482 |

#### 🔑 核心贡献
提出 **PathVQ**，一种基于向量量化（VQ）蒸馏的病理学基础模型重建方法，解决 WSI 超高分辨率带来的存储和计算开销挑战。PathVQ 通过在 patch 特征上进行 VQ 蒸馏，使用离散索引和解码器高效压缩空间 patch token，将 token 维度从 1024 降至 16，实现 **64 倍压缩率**，同时保持重建保真度。核心创新是多尺度 VQ（MSVQ）作为自监督学习监督信号，为下游 WSI 任务提供富含空间信息的表示。

#### 🔬 方法论
- **向量量化蒸馏**：将高维 patch 特征压缩为离散索引，通过量化解码器重建原始特征
- **多尺度 VQ（MSVQ）**：在多个尺度进行量化，增强 VQ 重建性能并作为 SSL 监督信号
- **渐进卷积模块**：基于量化 patch 特征开发，用于提取富含空间信息的下游表示
- **无缝切片级预训练目标**：将 MSVQ 与切片级 SSL 结合，优化 WSI 级别任务性能

#### 📊 实验结果
- Token 压缩率达 **64 倍**（1024 → 16 维度），显著降低存储和计算开销
- 保持重建保真度的同时，在 WSI 分析任务上达到最优性能
- 多尺度量化比单尺度方案重建质量更高
- 富含空间信息的表示对下游任务（分类、分割）有显著提升

#### ⚠️ 局限性
- 量化过程中可能丢失部分细粒度信息，对极细粒度诊断任务的影响需进一步评估
- 离散表示可能限制梯度传播的有效性，需要精心设计的蒸馏策略
- 压缩率与下游性能的权衡在不同数据集上可能不同

#### 📝 Abstract（原文）
> Computational pathology and whole-slide image (WSI) analysis are pivotal in cancer diagnosis and prognosis. However, the ultra-high resolution of WSIs presents significant modeling challenges. Recent advances in pathology foundation models have improved performance, but most approaches rely on tile-level ViT's [CLS] token representations as slide-level inputs, discarding critical spatial details from patch tokens. Using all spatial patch tokens benefits WSI analysis but incurs nearly 200x storage and training cost overhead. To address this, we propose PathVQ, a vector quantization (VQ) distillation approach that compresses spatial patch tokens into discrete indices and a decoder for efficient reconstruction. PathVQ achieves a 64x compression rate (1024→16 dimensions) while maintaining reconstruction fidelity. As a core innovation, we introduce multi-scale VQ (MSVQ) that enhances VQ reconstruction performance and serves as an SSL supervision signal. Based on quantized patch features and MSVQ tile-level supervision objectives, we develop progressive convolutional modules for extracting spatially enriched representations for downstream WSI tasks.

#### 📝 摘要（中文翻译）
计算病理学和全切片图像（WSI）分析在癌症诊断和预后中起着至关重要的作用。然而，WSI 的超高分辨率带来了显著的建模挑战。近期病理学基础模型的进展提升了性能，但大多数方法依赖于 tile 级 ViT 的 [CLS] token 表示作为切片级输入，丢弃了 patch token 中关键的空间细节。使用所有空间 patch token 有利于 WSI 分析，但会产生近 200 倍的存储和训练成本开销。为解决此问题，我们提出 PathVQ，一种向量量化（VQ）蒸馏方法，将空间 patch token 压缩为离散索引和解码器进行高效重建。PathVQ 实现 64 倍压缩率（1024→16 维度），同时保持重建保真度。作为核心创新，我们引入多尺度 VQ（MSVQ）来增强 VQ 重建性能并作为 SSL 监督信号。基于量化 patch 特征和 MSVQ tile 级监督目标，我们开发渐进卷积模块来提取富含空间信息的下游 WSI 任务表示。

### 3. Benchmarking Computational Pathology Foundation Models For Semantic Segmentation

| 项目 | 内容 |
|------|------|
| **作者** | Lavish Ramchandani, Aashay Tinaikar, Dev Kumar Das, Rohit Garg, Tijo Thomas |
| **来源** | ISBI 2026 · arXiv:2602.18747 |
| **关键词** | 病理基础模型基准测试, 语义分割, 注意力图, XGBoost, 模型集成, CONCH, PathDino |
| **原文** | https://arxiv.org/abs/2602.18747 |

#### 🔑 核心贡献
提出首个针对病理组织学图像像素级语义分割任务的**系统性基准测试**，对 10 个病理基础模型在 4 个数据集上进行评估，涵盖形态学组织区域分割和细胞/核分割任务。方法利用基础模型的注意力图作为像素级特征，结合 XGBoost 分类器实现快速、可解释且无需微调的模型无关评估。实验结果表明 **CONCH**（视觉语言基础模型）表现最优，**CONCH + PathDino + CellViT** 特征拼接平均提升 **7.95%**，证明模型集成策略可更好泛化到多样化组织病理学分割任务。

#### 🔬 方法论
- **注意力图特征提取**：利用预训练病理 FM 的注意力图作为像素级特征，无需额外标注
- **XGBoost 分类器**：使用梯度提升决策树进行像素分类，实现快速推理和可解释性
- **10 个基础模型评估**：涵盖纯视觉模型（ViT、DINO、PathDino）和视觉语言模型（CLIP、CONCH）
- **多数据集验证**：4 个组织病理学数据集，覆盖组织区域和细胞/核分割任务

#### 📊 实验结果
- **CONCH** 在所有数据集中表现最优，视觉语言模型优势明显
- **PathDino** 紧随其后，纯视觉基础模型中排名最高
- **特征拼接集成**（CONCH + PathDino + CellViT）平均性能提升 **7.95%**
- 无需微调的评估方式验证了基础模型的零样本迁移能力

#### ⚠️ 局限性
- 注意力图特征可能不完全捕捉像素级语义，对边界区域的分割精度有限
- XGBoost 作为固定分类器可能限制特征利用的灵活性
- 不同组织类型和染色协议的泛化性需要更大规模验证

#### 📝 Abstract（原文）
> In recent years, foundation models such as CLIP, DINO, and CONCH have demonstrated remarkable domain generalization and unsupervised feature extraction capabilities. However, a systematic and independent evaluation of these models for pixel-level semantic segmentation tasks on histopathology images remains underexplored. We propose a robust benchmarking approach evaluating 10 foundation models across 4 histopathology datasets, covering morphological tissue region segmentation and cell/nuclei segmentation tasks. Our approach leverages attention maps from foundation models as pixel-level features, then employs XGBoost for classification, enabling rapid, interpretable, and fine-tuning-free model-agnostic evaluation. Our findings reveal that CONCH (vision-language foundation model) outperforms pure vision foundation models, with PathDino ranking second. Additionally, models trained on diverse histopathology cohorts capture complementary morphological representations, and concatenating their features yields superior segmentation performance. Specifically, CONCH + PathDino + CellViT feature concatenation surpasses individual models with an average improvement of 7.95%, demonstrating that foundation model ensembles can better generalize to diverse histopathology segmentation tasks.

#### 📝 摘要（中文翻译）
近年来，CLIP、DINO 和 CONCH 等基础模型在各种医学成像任务中展现了卓越的领域泛化能力和无监督特征提取能力。然而，针对病理组织学图像的像素级语义分割任务，对这些模型进行系统性和独立评估的研究仍然非常匮乏。本研究提出了一种稳健的基准测试方法，对 10 个基础模型在 4 个组织病理学数据集上进行评估，涵盖形态学组织区域分割和细胞/核分割任务。我们的方法利用基础模型的注意力图作为像素级特征，然后使用 XGBoost 进行分类，实现快速、可解释且无需微调的模型无关评估。我们的发现表明，CONCH（视觉语言基础模型）优于纯视觉基础模型，PathDino 排名第二。此外，在不同组织病理学队列上训练的模型捕捉到互补的形态学表征，拼接它们的特征可获得更优的分割性能。具体而言，CONCH + PathDino + CellViT 特征拼接比单个模型平均提升 7.95%，表明基础模型集成可更好泛化到多样化组织病理学分割任务。
