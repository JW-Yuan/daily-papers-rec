## 1. MUSE: Harnessing Precise and Diverse Semantics for Few-Shot Whole Slide Image Classification

| 项目 | 内容 |
|------|------|
| **作者** | Jiahao Xu, Sheng Huang, Xin Zhang, Zhixiong Nan, Jiajun Dong, Nankun Mu |
| **来源** | CVPR 2026 · arXiv:2602.20873 |
| **关键词** | 少样本WSI分类, 多视图语义增强, MoE, 大语言模型, 视觉语言模型, 病理图像分析 |
| **原文** | https://arxiv.org/abs/2602.20873 |
| **代码** | https://github.com/JiahaoXu-god/CVPR2026_MUSE |
| **架构图** | `papers/imgs/2026-05-04/MUSE.png` |

### 核心贡献
- 提出 **MUSE（随机多视图语义增强）** 框架，通过样本级细粒度语义增强和基于检索的多视图生成，解决现有视觉-语言方法中文本语义静态、缺乏样本级调整的问题
- 在三个WSI基准数据集上，少样本设置下持续优于现有视觉-语言基线方法

### 方法论
- **问题背景**：计算病理学中少样本全切片图像分类的核心挑战在于专家标注切片的极端稀缺性；现有视觉-语言方法引入大语言模型生成的文本语义，但将这些描述视为跨所有样本共享的静态类级先验，缺乏样本级精细化调整
- **样本级细粒度语义增强（SFSE）**：通过基于MoE的自适应视觉-语义交互，为每个样本生成专属的细粒度语义先验，提升视觉-语义对齐的精度与多样性
- **随机多视图模型优化（SMMO）**：为每一类病理构建由大语言模型生成的多描述知识库，在训练过程中基于样本级语义先验检索并随机整合多个匹配的文本视图，提升模型鲁棒性并缓解过拟合
- **实验验证**：在三个WSI基准数据集上的实验表明，MUSE在少样本设置下持续优于现有视觉-语言基线方法

### 实验结果
- 在三个WSI基准数据集的少样本分类任务上，性能均优于现有视觉-语言基线模型
- 通过SFSE和SMMO两大模块，同时提升了语义精度与多样性
- 方法在少样本场景下的泛化能力得到验证

### 局限性
- 方法依赖大语言模型生成的多描述知识库，LLM生成质量可能影响最终效果
- 在计算资源受限环境下的部署效率尚需进一步验证
- 目前仅在三种癌症类型（乳腺癌、肺癌等）的WSI数据集上验证，对其他癌种的泛化性有待评估

### Abstract
In computational pathology, few-shot whole slide image classification is primarily driven by the extreme scarcity of expert-labeled slides. Recent vision-language approaches have introduced textual semantics generated by Large Language Models (LLMs), yet treat these descriptions as static, class-level priors shared across all samples, lacking sample-specific refinement. This simultaneously limits both the diversity and the precision of vision-semantic alignment, hindering model generalization under limited supervision. To address this, we propose MUSE (stochastic Multi-view Semantic Enhancement), a framework that first optimizes semantic precision at the sample level, then enriches the semantic repository through retrieval-augmented multi-view generation. Specifically, MUSE comprises two core designs: Sample-specific Fine-grained Semantic Enhancement (SFSE), which generates exclusive fine-grained semantic priors for each sample via MoE-based adaptive vision-semantics interaction; and Stochastic Multi-view Model Optimization (SMMO), which first constructs an LLM-generated multi-description knowledge base for each pathology category, then retrieves and stochastically integrates multiple matching text views based on sample-level semantic priors during training. These dynamically selected texts serve as enhanced semantic supervision signals, optimizing the vision-language model stochastically to improve model robustness and alleviate overfitting. Experiments on three WSI benchmark datasets demonstrate that MUSE consistently outperforms existing vision-language baselines in few-shot settings, proving that effective few-shot pathology learning requires not only richer semantic sources, but also active, sample-aware optimization of these semantics. Our code is publicly available at: https://github.com/JiahaoXu-god/CVPR2026_MUSE.

### 摘要（中文翻译）
在计算病理学中，少样本全切片图像（WSI）分类的核心挑战在于专家标注切片的极端稀缺性。近期的视觉-语言方法虽然引入了大语言模型生成的文本语义，但将这些描述视为跨所有样本共享的静态类级先验，缺乏样本级的精细化调整，这同时限制了视觉-语义对齐的多样性与精度，阻碍了有限监督下的模型泛化能力。为解决这一问题，我们提出了**随机多视图语义增强（MUSE）**框架，该框架首先通过样本级适配优化语义精度，再通过检索增强的多视图生成丰富语义储备。具体而言，MUSE包含两个核心设计：样本级细粒度语义增强（SFSE）：通过基于MoE的自适应视觉-语义交互，为每个样本生成专属的细粒度语义先验；随机多视图模型优化（SMMO）：首先为每一类病理构建由大语言模型生成的多描述知识库，在训练过程中基于样本级语义先验检索并随机整合多个匹配的文本视图。这些动态选择的文本作为增强的语义监督信号，对视觉-语言模型进行随机优化，提升模型鲁棒性并缓解过拟合问题。在三个WSI基准数据集上的实验表明，MUSE在小样本设置下持续优于现有视觉-语言基线方法，证明有效的小样本病理学习不仅需要更丰富的语义来源，还需要对这些语义进行主动的、样本感知的优化。我们的代码已公开在：https://github.com/JiahaoXu-god/CVPR2026_MUSE。

## 2. Momentum Memory for Knowledge Distillation in Computational Pathology

| 项目 | 内容 |
|------|------|
| **作者** | Yongxin Guo, Hao Lu, Onur C. Koyun, Zhengjie Zhu, Muhammet Fatih Demir, Metin Nafi Gurcan |
| **来源** | CVPR 2026 · arXiv:2602.21395 |
| **关键词** | 知识蒸馏, 计算病理学, 跨模态学习, 动量记忆, 基因组学, 组织学图像 |
| **原文** | https://arxiv.org/abs/2602.21395 |
| **代码** | https://github.com/YongxinGuo/MoMKD |
| **架构图** | `papers/imgs/2026-05-04/MoMKD.png` |

### 核心贡献
- 提出 **MoMKD（动量记忆知识蒸馏）**，通过动量更新的记忆模块跨批次聚合基因组学与组织病理学信息，解决现有知识蒸馏方法依赖批次内局部对齐导致的训练不稳定问题
- 对基因组分支与组织学分支的梯度进行解耦，防止训练过程中基因组信号主导组织学特征学习，同时消除推理阶段的模态鸿沟问题

### 方法论
- **问题背景**：整合基因组学与组织病理学的多模态学习在癌症诊断中展现出强大潜力，但其临床转化受限于配对的组织学-基因组学数据稀缺；知识蒸馏（KD）提供了一种实用解决方案，可将基因组监督信息迁移至组织病理学模型，实现仅用组织学数据即可完成准确推理；然而，现有知识蒸馏方法依赖批次内局部对齐，因批次内比较有限会导致训练不稳定，最终降低性能
- **动量更新记忆模块**：MoMKD引入动量更新的记忆模块，跨多个训练批次聚合基因组学与组织病理学特征信息，为每个mini-batch提供更丰富的监督上下文，提升训练稳定性与性能
- **梯度解耦设计**：对基因组分支与组织病理学分支的梯度进行解耦，避免训练过程中基因组模态的强监督信号主导组织病理学分支的特征学习，同时解决推理阶段（仅使用组织学数据）的模态鸿沟问题
- **推理特性**：训练完成后，模型仅需输入组织病理学数据即可完成推理，无需配对基因组数据，降低临床落地对多模态数据的依赖

### 实验结果
- 在 **TCGA-BRCA基准数据集**（包含HER2、PR、ODX分类任务）和独立内部测试集上开展大量实验
- MoMKD持续优于当前最优的多示例学习（MIL）与多模态知识蒸馏基线方法
- 在组织学单模态推理场景下具备优异的性能与泛化性

### 局限性
- 方法需要同时获取基因组学和组织病理学配对数据进行训练，临床数据获取成本较高
- 动量记忆模块的超参数（动量更新率等）对不同癌症类型的敏感性尚需系统研究
- 在跨癌症泛化场景下的性能衰减问题有待进一步探索

### Abstract
Multimodal learning that integrates genomics and histopathology has shown strong potential in cancer diagnosis, yet its clinical translation is hindered by the limited availability of paired histology-genomics data. Knowledge distillation (KD) offers a practical solution by transferring genomic supervision into histopathology models, enabling accurate inference using histology alone. However, existing KD methods rely on batch-local alignment, which introduces instability due to limited within-batch comparisons and ultimately degrades performance. To address these limitations, we propose Momentum Memory Knowledge Distillation (MoMKD), a cross-modal distillation framework driven by a momentum-updated memory. This memory aggregates genomic and histopathology information across batches, effectively enlarging the supervisory context available to each mini-batch. Furthermore, we decouple the gradients of the genomics and histology branches, preventing genomic signals from dominating histology feature learning during training and eliminating the modality-gap issue at inference time. Extensive experiments on the TCGA-BRCA benchmark (HER2, PR, and ODX classification tasks) and an independent in-house testing dataset demonstrate that MoMKD consistently outperforms state-of-the-art MIL and multimodal KD baselines, delivering strong performance and generalization under histology-only inference. Overall, MoMKD establishes a robust and generalizable knowledge distillation paradigm for computational pathology.

### 摘要（中文翻译）
整合基因组学与组织病理学的多模态学习在癌症诊断中展现出强大潜力，但其临床转化受限于配对的组织学-基因组学数据稀缺。知识蒸馏（KD）提供了一种实用解决方案，可将基因组监督信息迁移至组织病理学模型，实现仅用组织学数据即可完成准确推理。然而，现有知识蒸馏方法依赖批次内局部对齐，因批次内比较有限会导致训练不稳定，最终降低性能。为解决上述局限，我们提出**动量记忆知识蒸馏（MoMKD）**，一种由动量更新记忆驱动的跨模态蒸馏框架。该记忆模块可跨批次聚合基因组学与组织病理学信息，有效扩大每个mini-batch可用的监督上下文。此外，我们对基因组分支与组织学分支的梯度进行解耦，防止训练过程中基因组信号主导组织学特征学习，同时消除推理阶段的模态鸿沟问题。我们在TCGA-BRCA基准数据集（包含HER2、PR、ODX分类任务）和独立内部测试集上开展了大量实验，结果表明MoMKD持续优于当前最优的多示例学习（MIL）与多模态知识蒸馏基线方法，在组织学单模态推理场景下具备优异的性能与泛化性。总体而言，MoMKD为计算病理学构建了一种鲁棒、可泛化的知识蒸馏范式。

## 3. UNIStainNet: Foundation-Model-Guided Virtual Staining of H&E to IHC

| 项目 | 内容 |
|------|------|
| **作者** | Jillur Rahman Saurav, Thuong Le Hoai Pham, Pritam Mukherjee, Paul Yi, Brent A. Orr, Jacob M. Luber |
| **来源** | CVPR 2026 · arXiv:2603.12716 |
| **关键词** | 虚拟染色, IHC染色, H&E染色, 病理基础模型, SPADE-UNet, 计算病理学 |
| **原文** | https://arxiv.org/abs/2603.12716 |
| **代码** | https://github.com/facevoid/UNIStainNet |
| **架构图** | `papers/imgs/2026-05-04/UNIStainNet.png` |

### 核心贡献
- 提出 **UNIStainNet**，首次将冻结的病理基础模型UNI的密集空间token作为SPADE-UNet的条件信号，为H&E到IHC的染色转换过程提供组织层面的语义引导
- 通过可学习的染色嵌入，使单个统一模型可同时支持多个IHC标记物（HER2、Ki67、ER、PR）的生成，无需像传统方法那样为每个IHC染色单独训练模型

### 方法论
- **问题背景**：虚拟免疫组化（IHC）染色可从苏木精-伊红（H&E）图像生成，通过直接从常规病理切片提供初步分子层面见解来加速诊断，在组织样本有限时减少重复切片的必要性；现有方法通过对比学习目标、原型匹配或域对齐技术提升生成图像的真实性，但生成器本身未获得病理学基础模型的直接引导
- **SPADE-UNet架构**：采用SPADE-UNet作为骨干网络，输入条件为冻结的病理学基础模型UNI输出的密集空间token，利用基础模型预训练的语义特征，为染色转换提供组织层面的语义引导
- **不对齐感知损失套件**：设计专用不对齐感知损失套件，针对性保留染色结果的定量准确性，避免转换过程中出现染色定量偏差
- **多任务特性**：引入可学习的染色嵌入模块，使单一模型可同时支持多个不同IHC标记物的生成，无需像传统方法那样为每个IHC染色单独训练独立模型

### 实验结果
- 在 **MIST数据集**上，单一模型对HER2、Ki67、ER、PR四种常见临床IHC染色的分布指标均达到SOTA（现有最优方法通常需为每个染色单独训练模型）
- 在 **BCI数据集**上也取得最优的分布指标
- 组织类型分层失败分析表明，模型剩余误差具有系统性，主要集中在非肿瘤组织区域

### 局限性
- 方法依赖冻结的病理基础模型UNI，UNI模型的可及性和许可可能需要考虑
- 在非肿瘤组织区域的虚拟染色精度仍有提升空间
- 当前仅在HER2、Ki67、ER、PR四种IHC标记物上验证，对其他标记物的泛化性有待评估

### Abstract
Virtual immunohistochemistry (IHC) staining can be generated from hematoxylin and eosin (H&E) images, accelerating diagnosis by providing preliminary molecular insights directly from routine pathology slides and reducing the need for repeated sectioning when tissue samples are limited. Existing methods enhance the realism of generated images through contrastive learning objectives, prototype matching, or domain alignment techniques, yet the generator itself receives no direct guidance from pathology foundation models. We present UNIStainNet, a SPADE-UNet conditioned on dense spatial tokens from a frozen pathology foundation model (UNI), providing tissue-level semantic guidance for stain translation. The model employs a misalignment-aware loss suite that preserves staining quantification accuracy, while learnable stain embeddings allow a single unified model to generate multiple IHC markers simultaneously. On the MIST dataset, UNIStainNet achieves state-of-the-art distribution metrics for all four stains (HER2, Ki67, ER, PR) with a single unified model, whereas existing methods typically require separate models trained for each stain. On the BCI dataset, it also attains the best distribution metrics. Failure analysis by tissue type reveals that the model's remaining error is systematic and concentrated in non-tumor tissue regions.

### 摘要（中文翻译）
虚拟免疫组化（IHC）染色可从苏木精-伊红（H&E）图像生成，通过直接从常规病理切片提供初步分子层面见解来加速诊断，在组织样本有限时减少重复切片的必要性。现有方法通过对比学习目标、原型匹配或域对齐技术提升生成图像的真实性，但生成器本身未获得病理学基础模型的直接引导。我们提出UNIStainNet，这是一种以冻结态病理学基础模型（UNI）输出的密集空间token为条件的SPADE-UNet架构，为染色转换过程提供组织层面的语义引导。模型采用不对齐感知损失套件，保留染色定量的准确性；同时通过学习得到的染色嵌入，让单个统一模型可同时支持多个IHC标记物的生成。在MIST数据集上，UNIStainNet仅用单一统一模型，就在HER2、Ki67、ER、PR全部四种染色的分布指标上达到当前最优（SOTA）水平，而现有方法通常需为每个染色单独训练模型。在BCI数据集上，它也取得了最优的分布指标。组织类型分层失败分析显示，模型剩余误差具有系统性，集中在非肿瘤组织区域。
