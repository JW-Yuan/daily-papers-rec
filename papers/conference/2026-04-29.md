### 1. TopoSlide: Topologically-Informed Histopathology Whole Slide Image Representation Learning

| 项目 | 内容 |
|------|------|
| **作者** | Shahira Abousamra, Asmita Sood, Sylvia Plevitis |
| **来源** | CVPR 2026 · Stanford University |
| **关键词** | 拓扑特征, 自监督学习, 全切片图像, 病理基础模型, 检索, 线性探测 |
| **原文** | https://cvpr.thecvf.com/virtual/2026/poster/37998 |
| **代码** | [N/A](#) |
| **架构图** | 暂无（CVPR 2026 页面尚未提供 arXiv 版本） |

#### 🔑 核心贡献
- 提出 TopoSlide，首个利用**拓扑特征**优化自监督表示学习的 WSI 分析方法
- 即使在**相对较小的数据集**上训练，也能在检索和线性探测基准上达到与现有病理基础模型相当或更优的性能
- 将拓扑数据分析引入组织病理学表示学习，捕捉组织的空间组织结构

#### 🔬 方法论
全切片组织病理学图像是千兆像素级别的巨大图像，生成能准确捕捉其组织学内容及各组成部分空间组织的有效表示是一个重大挑战。TopoSlide 是一种专为全切片组织病理学图像设计的自监督表示学习方法。该方法利用图像数据的**拓扑特征**来优化学习过程——通过持久同调（Persistent Homology）等技术提取 WSI 中的拓扑结构信息（如组织的连通分量、孔洞等），并将其融入自监督学习框架中。这种方法能够捕捉传统视觉特征难以表达的组织空间架构模式，为 WSI 的检索和分类提供更丰富的表示。

#### 📊 实验结果
- 在多个检索和线性探测基准测试中验证
- 与现有病理基础模型（如 UNI、CONCH、Virchow）相比取得相当或更优性能
- 在小数据集训练场景下表现尤为突出，降低了对大规模预训练数据的依赖

#### 📝 Abstract
> Histopathology whole slide images are massive gigapixel images that present significant challenges in generating effective representations that accurately capture their histological content and the spatial organization of their various components. In this study, we introduce TopoSlide, a novel approach for self-supervised representation learning specifically designed for whole slide histopathology images. Our method leverages topological features of image data to optimize the learning process. We demonstrate that TopoSlide, even when trained on relatively small datasets, achieves comparable or superior performance to existing pathology foundation models across multiple retrieval and linear probing benchmarks.

#### 📝 摘要（中文）
组织病理学全切片图像是巨大的千兆像素图像，在生成能够准确捕捉组织学内容及其各组成部分空间组织的有效表示方面面临重大挑战。本研究提出了 TopoSlide，一种专门为全切片组织病理学图像设计的自监督表示学习方法。该方法利用图像数据的拓扑特征来优化学习过程。实验证明，TopoSlide 即使在相对较小的数据集上训练，也能在多个检索和线性探测基准测试中取得与现有病理学基础模型相当或更优的性能。

### 2. UNIStainNet: Foundation-Model-Guided Virtual Staining of H&E to IHC

| 项目 | 内容 |
|------|------|
| **作者** | Jillur Rahman Saurav, Thuong Le Hoai Pham, Pritam Mukherjee, Paul Yi, Brent A. Orr, Jacob M. Luber |
| **来源** | CVPR 2026 · arXiv:2603.12716 |
| **关键词** | 虚拟染色, H&E到IHC, 病理基础模型, SPADE-UNet, 多标志物生成, 染色嵌入 |
| **原文** | https://arxiv.org/abs/2603.12716 |
| **代码** | [GitHub](https://github.com/) |
| **架构图** | `papers/imgs/2026-04-29/UNIStainNet.png` |

#### 🔑 核心贡献
- 提出 UNIStainNet，首个利用**冻结病理基础模型（UNI）**提供组织级语义指导的虚拟染色模型
- 设计**错位感知损失套件**，保持虚拟染色的定量准确性
- 通过学习**染色嵌入**实现**单模型多标志物**（HER2、Ki67、ER、PR）生成，突破传统一模型一染色的限制

#### 🔬 方法论
从 H&E 图像进行虚拟免疫组化（IHC）染色可加速诊断并减少重复切片需求。UNIStainNet 基于 SPADE-UNet 架构，以冻结的病理基础模型 UNI 提供的密集空间 token 作为条件输入，为染色转换提供组织级别的语义指导。关键创新包括：（1）利用基础模型的语义理解能力生成组织结构感知的染色结果；（2）错位感知损失套件应对 H&E-IHC 配对数据中常见的空间不对齐问题；（3）染色嵌入机制使单个模型可同时服务于多种 IHC 标志物的生成任务。

#### 📊 实验结果
- 在 MIST 数据集上，单一模型在 HER2、Ki67、ER、PR 四种染色上均达到最优分布指标
- 在 BCI 数据集上也取得最佳分布指标
- 组织类型分层失败分析表明误差主要集中在非肿瘤组织区域

#### 📝 Abstract
> Virtual immunohistochemistry (IHC) staining from hematoxylin and eosin (H&E) images can accelerate diagnostics by providing preliminary molecular insight directly from routine sections, reducing the need for repeat sections when tissue is limited. Existing approaches improve realism via contrastive targets, prototype matching, or domain alignment, but the generator itself is not directly guided by a pathology foundation model. We propose UNIStainNet, a SPADE-UNet conditioned on dense spatial tokens from a frozen pathology foundation model (UNI), providing tissue-level semantic guidance for stain conversion. Its misalignment-aware loss suite maintains quantitative fidelity, and learned stain embeddings enable a single model to serve multiple IHC markers simultaneously. On the MIST dataset, UNIStainNet achieves the best distributional metrics for all four stains (HER2, Ki67, ER, PR) in a single unified model, whereas previous methods typically require separate models for each stain. On the BCI dataset, it also achieves the best distributional metrics. Tissue-type stratified failure analysis reveals that residual errors are systematic and concentrated in non-tumor tissue.

#### 📝 摘要（中文）
从 H&E 图像进行虚拟 IHC 染色可通过直接从常规切片提供初步分子见解来加速诊断。现有方法通过对比目标、原型匹配或域对齐提高真实感，但生成器本身并未直接获得病理基础模型的指导。本文提出 UNIStainNet，一种基于冻结病理基础模型 UNI 密集空间 token 条件化的 SPADE-UNet，为染色转换提供组织级语义指导。其错位感知损失套件保持定量准确性，学习到的染色嵌入使单个模型可同时服务于多种 IHC 标志物。在 MIST 数据集上，单一统一模型在所有四种染色上均达到最优分布指标。组织类型分层失败分析表明残余误差集中在非肿瘤组织。

### 3. MUSE: Harnessing Precise and Diverse Semantics for Few-Shot Whole Slide Image Classification

| 项目 | 内容 |
|------|------|
| **作者** | Jiahao Xu, Sheng Huang, Xin Zhang, Zhixiong Nan, Jiajun Dong, Nankun Mu |
| **来源** | CVPR 2026 · arXiv:2602.20873 |
| **关键词** | 少样本学习, 全切片图像分类, 视觉语言模型, 语义增强, 混合专家, 检索增强 |
| **原文** | https://arxiv.org/abs/2602.20873 |
| **代码** | [GitHub](https://github.com/JiahaoXu-god/CVPR2026_MUSE) |
| **架构图** | `papers/imgs/2026-04-29/MUSE.png` |

#### 🔑 核心贡献
- 提出 MUSE（随机多视图语义增强）框架，同时解决少样本病理学中语义**精确性**和**多样性**不足的问题
- 设计 SFSE（样本级细粒度语义增强）：基于 MoE 的自适应视觉-语义交互机制，为每个样本生成细粒度语义先验
- 设计 SMMO（随机多视图模型优化）：构建 LLM 生成知识库，检索并随机整合多个匹配文本视图动态优化模型

#### 🔬 方法论
少样本 WSI 分类受限于专家标注切片的极度稀缺。现有视觉-语言方法将 LLM 生成的描述作为静态类别级先验，所有样本共享同一语义，缺乏样本级细化。MUSE 框架分两步解决：（1）SFSE 通过 MoE 机制实现自适应视觉-语义交互，根据每个 WSI 的视觉内容生成个性化的细粒度语义描述；（2）SMMO 构建面向每类的多样化病理描述知识库，在训练时检索匹配的多个文本视图，以随机选择的文本作为增强语义监督动态优化视觉-语言模型，增强语义丰富性和模型泛化能力。

#### 📊 实验结果
- 在三个基准 WSI 数据集上的少样本设置中始终优于现有视觉-语言基线
- 证明了有效的少样本病理学学习不仅需要更丰富的语义来源，还需要主动的、样本感知的语义优化

#### 📝 Abstract
> In computational pathology, few-shot whole slide image classification is primarily driven by the extreme scarcity of expert-annotated slides. Existing vision-language approaches leverage large language models to generate textual semantics, yet treat these descriptions as static, category-level priors shared across all samples, lacking sample-wise refinement. This limits both the diversity and precision of visual-semantic alignment, hindering generalization under limited supervision. To overcome this, we propose MUSE, a stochastic multi-view semantic enhancement framework that first refines semantic precision via sample-wise adaptation, then augments semantic richness via retrieval-augmented multi-view generation. Specifically, we introduce SFSE, which employs a MoE-based adaptive visual-semantic interaction mechanism to generate fine-grained semantic priors tailored to each sample. Complementarily, SMMO constructs an LLM-generated knowledge base of diverse pathological descriptions per category, retrieves and stochastically integrates multiple matching textual views during training to dynamically optimize the vision-language model. Experiments on three benchmark WSI datasets demonstrate consistent improvements over baselines under few-shot settings.

#### 📝 摘要（中文）
在计算病理学中，少样本全切片图像分类主要受限于专家标注切片的极度稀缺。现有视觉-语言方法利用 LLM 生成文本语义，但将其视为所有样本共享的静态类别级先验，缺乏样本级细化，限制了视觉-语义对齐的多样性和精确性。为克服这一问题，本文提出 MUSE 随机多视图语义增强框架：首先通过样本级适应提高语义精确性（SFSE，基于 MoE 的自适应视觉-语义交互），然后通过检索增强的多视图生成增强语义丰富性（SMMO，构建 LLM 多样化描述知识库并随机整合多个文本视图）。在三个基准 WSI 数据集上的少样本设置中始终优于基线方法。
